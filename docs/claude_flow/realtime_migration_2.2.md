# 1. Initialize the system
  npx claude-flow hive-mind init

  # 2. Spawn a swarm for a task
  npx claude-flow hive-mind spawn "Build a REST API with authentication"

  # 3. Check status
  npx claude-flow hive-mind status

  # 4. Advanced options
  npx claude-flow hive-mind spawn "Research design patterns" --queen-type adaptive --max-workers 5

  When to Use It

  - Complex, multi-faceted projects requiring different expertise
  - Tasks needing collaborative problem-solving
  - When you want multiple AI perspectives on a solution
  - Large-scale refactoring or architectural decisions
